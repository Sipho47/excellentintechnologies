# Excellent In Technologies

Welcome to my website repository hosted on GitHub Pages.

## Description

This repository contains website files, HTML5 exercises, and application examples.

## Loan System

The repository now includes a Flask/PostgreSQL loan management app in [`loan_system`](loan_system/).

The loan system supports admin login, CRUD loan records, search, pagination, repayment calculations, and dashboard totals for capital and projected profit. See [`loan_system/README.md`](loan_system/README.md) for setup, database, and test instructions.

## Live Website

## Structure

The repository includes:

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Sipho47/excellentintechnologies.git
   ```
2. Open `index.html` in your browser to view the site locally.
3. To run the loan app, follow the instructions in `loan_system/README.md`.
